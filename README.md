# Guess Word
[![Travis build](https://travis-ci.org/philiplambok/guess-word.svg?branch=master)](https://travis-ci.org/philiplambok/guess-word)


### Stack

- Ruby 2.5.3
- Rails 6.0.0
- PostgreSQL 9.5.15
- Node.js v8.15.0
- Vue.js 2.6.10

### Usage

```rb
$> bundle install
$> yarn install
$> rspec
$> rails db:migrate && rails db:seed
$> rails s -p 3000
```
